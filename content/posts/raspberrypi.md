I have a work laptop now, so there's a possibility that I'll have to work and do non-work computational things in the same day. Ideally, I'd like to keep my work devices clean of any non-work files. To do this, I decided to pick up a raspberry pi Nano w. The thing with these little devices, is the capability of networking over USB. So, if I connect it to the laptop and configure it, I can easily SSH into it. So now I can do school assignemnts, research (the light kind), write (I'm actually writing this post from it), etc. For now, this suffices. However, I may find myself needing to use zoom or some conferencing application for classes. I haven't thought much about that scenario, but I imagine using VNC along with a lightweight WM to run only a browser might work. I'll think about it, maybe try it and share the results.  
  
#### Observation
I coincidentially connected a RPi 3 (via USB to Serial Adapter) and the RPi Nano w to the same computer and set up the network. However, when I tried to SSH, I was accessing the RPi 3 instead of the RPi Nano w, which is unexpected. Was the RPi 3 doing networking over Serial? I'll try it again in an isolated environment and share the results. 
All in all, I'm happy with it for now.  
  
That is all.
